문제 4.¶

국회의원 데이터를 활용한 군집 분석¶

1. 서론¶

1) 분석 목표¶

국회의원 법안 발의 건수 데이터를 활용하여 의원들의 군집을 파악하고 각 군집에 대한 의미를 파악한다.

2) 분석에 사용한 데이터¶

제 20대 국회의원 중 141명의 개별 의원들이 발의한 법안 건수가 입력된 자료를 사용하였다.

자료의 대각 원소는 해당 의원이 단독으로 발의한 법안의 수 이며, 나머지 원소는 두 의원이 함께 발의한 법안의 수 이다.

2. 본론¶

1) 분석 방법 소개¶

분석에는 군집 분석을 사용하였고 구체적으로 사용한 군집 분석 방법은 다음과 같다.

(1) Hierarchical agglomerative clustering : single linkage, complete linkage, average linkage¶

→ 관측 값들을 특정 기준에 따라 차례로 묶거나 차례로 분리해가면서 적당한 군집의 수를 찾아내는 방법

(2) Partitioning clustering : k-medoids¶

→ k-means 군집 분석의 일종으로 k-means가 임의의 좌표를 중심점으로 잡는 반면, 
  k-medoids는 실제 점 하나를 중심점으로 잡아 계산을 수행한다.
  이 방법을 활용하여 의원들에 대한 군집 분석의 결과를 도출했다.

적절한 군집의 수를 판단하기 위해 다음과 같은 계층적 군집 분석 과정들을 적용해 보았다.

② Partitioning clustering (k-medoids)¶

partyFreq = function(data, cluster)
{
  outPut = NULL
  for (i in 1:length(cluster))
  {
    partySum = c()
    g = sum(data[,colnames(data) == names(cluster[i])][which(data$party == '국민의당')])
    d = sum(data[,colnames(data) == names(cluster[i])][which(data$party == '더불어민주당')])
    f = sum(data[,colnames(data) == names(cluster[i])][which(data$party == '자유한국당')])
    b = sum(data[,colnames(data) == names(cluster[i])][which(data$party == '바른정당')])
    j = sum(data[,colnames(data) == names(cluster[i])][which(data$party == '정의당')])
    m = sum(data[,colnames(data) == names(cluster[i])][which(data$party == '무소속')])
    partySum = c(g, d, f, b, j, m) 
    names(partySum) = c('국민의당', '더불어민주당', '자유한국당', '바른정당', '정의당', '무소속')
    
    outPut = rbind(outPut, partySum)
    rownames(outPut)[i] = names(cluster[i])
  }
  print('아래는 각 의원의 정당별 협업 정도 데이터프레임')
  print(outPut)
  result = colSums(outPut)/length(cluster)
  result2 = transform(result)
  result2 = cbind(rownames(result2), result2)
  colnames(result2) = c('Party', 'Proposition')
  pie<- ggplot(result2, aes(x='', y=Proposition, fill=Party))+
    geom_bar(width = 1, stat = "identity") + coord_polar("y", start = 0)
  plot(pie)
  return(result)
}

3. 결론¶

1) 분석 결과¶

(1) 공동 법안 발의의 패턴¶

국회의원들의 법안 발의 건수 데이터를 활용하여 군집 분석을 시행 해 본 결과, 공동 법안 발의를 높은 비율로 시행한 국회의원들 간의 군집을 여러 방법으로 도출해 낼 수 있었다. 이 과정에서 형성된 군집들은 대부분 같은 당에 속해 있는 국회의원들 위주로 군집이 형성된다는 사실을 알 수 있었고, 이러한 모습을 벗어나는 의원들은 거의 나타나지 않았다. 따라서 현재 제 20대 국회의원들의 공동 법안 발의의 패턴은 주로 당 위주로, 당 내의 의원들 끼리 함께 협업하여 법안을 발의 한다는 사실을 도출할 수 있었다.

(2) 다른 당과의 협조 패턴¶

두 번째로, 자신의 당을 제외한 다른 당 의원들과 얼마나 공동 법안 발의를 했는지에 대해 분석을 해 본 결과, 해당 의원들이 다른 당의 어떤 의원과 그리고 어떤 당과 주로 공동 법안 발의를 하는지를 알 수 있었다. 이 결과 더불어민주당에 속해있는 의원들은 나머지 당 중 국민의당에 속해있는 의원들고 주로 공동법안을 발의하고, 마찬가지로 국민의당에 속해있는 의원들은 더불어민주당과 주로 공동법안을 발의하는 것을 알 수 있었다. 또한 자유한국당과 바른정당도 더불어민주당과 국민의당의 관계처럼 서로 간의 공동 법안 발의하는 비율이 다른 당에 비해 높다는 사실을 파악했다. 즉, 진보적인 성향을 가지는 더불어민주당과 국민의당이 함께 법안을 발의하는 경우가 많고, 보수적인 성향을 가지는 자유한국당과 바른정당이 함께 법안을 발의하는 경우가 많다는 사실이며 이는 우리가 통상적으로 생각하는 사실과도 일치한다.

2) 분석의 한계점¶

(1) 중복된 법안에 대한 정보 부족¶

분석을 토대로, 위와 같이 국회의원들을 군집화 함으로써 어떤 의원이 어떤 군집에 속하고, 또 자신의 당을 제외한 다른 어떤 당과 공동 법안 발의를 많이 하는지를 파악할 수 있었다. 하지만 주어진 데이터의 경우 단순하게 해당 의원이 자기 혼자 발의한 법안의 건수가 몇 건이고 타 의원과 함께 발의한 건수가 몇 건인지만 알 수 있을 뿐, 해당 법안에 얼마나 많은 사람들이 공동 법안 발의를 했는지에 대한 정보가 부족하였다. 따라서 데이터를 row 또는 columm으로 더했을 때, 나오는 수치가 개인발의, 공동발의를 포함해서 해당 의원이 총 발의한 건수로 보았던 점은 일부분 정보가 왜곡되었을 가능성을 무시할 수 없었다. 예를 들어 한 법안에 4명이 함께 공동발의를 한 경우 한 row에서 법안 발의 건수로 잡히는 수치는 1건이 아닌 4건이 되기 때문이다. 즉, 중복된 법안이 어떠한 형식으로 분포되어 있고 중복 되지 않은 법안은 몇 건인지에 대한 정보가 추가적으로 주어진다면 조금 더 정확한 데이터 분석이 가능할 것으로 판단된다.

(2) 추가적인 정보 부족¶

또한, 해당 국회의원들이 발의한 법안들이 어떠한 특성을 지니는 법안인지에 대한 정보가 있다면, 더욱 다양한 군집 분석을 도출할 수 있을 것이라고 생각된다. 예를 들어, 교육에 대한 법안/경제에 대한 법안/안보에 대한 법안 등 법안의 카테고리가 어디에 속해있는지에 대한 정보가 있다면, 그 중에서는 보수/진보 정당의 관계를 초월한 특성을 가지는 법안도 있고, 보수/진보 정당의 관계를 뚜렷하게 구분하는 법안도 있을 것이다. 또한 의원에 따라 같은 보수/진보 정당이라고 해도 법안에 공감하는 정도가 개개인마다 다를 것인데 이러한 특성의 패턴도 알 수 있다면 비교적 수월하게 군집을 분류 할수 있을 것이라고 생각된다. 따라서 단순히 법안을 발의한 건수가 몇 건인지에 대한 수치 정보에 추가적으로 해당 의원이 어떤 카테고리에 속하는 법안을 주로 발의했는지에 대한 정보가 추가된다면, 국회의원들을 군집으로 분류하는데에 중요한 역할을 하는 변수로 사용할 수 있을 것이라고 판단된다.
